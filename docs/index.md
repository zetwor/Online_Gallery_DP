# Персоналізована онлайн-галерея

## Опис проєкту
Це вебзастосунок для фотографів, який дозволяє створювати фотоальбоми, завантажувати зображення та надавати клієнтам доступ до них за посиланням.

## Основні можливості
- створення альбомів
- завантаження фотографій
- перегляд фото клієнтом
- вибір улюблених зображень
- зворотний зв’язок

## Використані технології
- HTML, CSS, JavaScript
- Firebase
- EmailJS
